Stack Overflow User Search

Form Input Info

Tag: This input controls what user skill is being searched, stack overflow uses tags when asking questions on the site. Any upvotes on questions and answers will affect the users score related to the tag.

Pages: This controls how many pages the api will query. Each page contains 100 user items that will be obtained and filtered. The higher the page count the longer the search will take.

Location: This filter option allows to filter the list of users by the user location listed on each users profile.

Reputation: Each user on stack overflow has a reputation score based on total upvotes and downvotes across the site. This allows a minimum requirement to be set.
